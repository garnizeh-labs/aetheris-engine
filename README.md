# Aetheris Engine

The authoritative simulation engine for the Aetheris ecosystem. Features high-performance spatial partitioning, interest management, priority-based replication channels, and multi-transport support.

## Documentation

Full documentation for the engine architecture and design can be found at: [docs.aetheris.io/engine](https://docs.aetheris.io/engine)

## Key Features

- **Zero-Trust Simulation**: Authoritative server with passive client observation.
- **Multi-Transport**: Support for Renet (UDP), Quinn (QUIC), and WebTransport.
- **Scaling Architecture**: Native support for room instances and planned federation.
