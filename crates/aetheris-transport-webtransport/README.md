# aetheris-transport-webtransport

WebTransport implementation for the Aetheris Engine.

## Overview

Provides a browser-native communication layer using the WebTransport protocol. Built on top of `wtransport`, it offers a modern alternative to WebSockets with better performance for real-time applications.
