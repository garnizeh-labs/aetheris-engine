# aetheris-transport-webtransport

WebTransport implementation for the Aetheris Engine.

## Overview

Provides a browser-native communication layer using the WebTransport protocol. Built on top of `wtransport` (QUIC/HTTP3), it offers a modern alternative to WebSockets with better performance for real-time applications.

## Technical Specifications

- **Role**: Browser-native Transport
- **Capabilities**: HTTP3 stream integration, self-signed certificate generation, and SHA-256 fingerprinting for Chrome/Firefox compliance.
